Flow Cell 操作：Wash 與 Preserve

完成 run 後，以 EXP-WSH004 清洗 flow cell 並以 Storage Buffer 保存於 2–8°C。
註釋定位

適用情境

情境 建議
當天不再跑第二個 library Wash 後加入 Storage Buffer 保存
Flow cell 仍有可用 pores 優先考慮 wash / preserve，保留後續再使用機會
Run 結束但無法立即清洗 越快越好；若無法當天清洗，維持 flow cell 在 device 上，隔天儘速處理
SOP 化紀錄需求 每次 wash / preserve 均需記錄 flow cell ID、時間、操作者、pore 數與異常

試劑與設備

類別 項目
Kit Flow Cell Wash Kit（EXP-WSH004 或 EXP-WSH004-XL）
Reagents Wash Mix（WMX）、Wash Diluent（DIL）、Storage Buffer（S）
Consumables 1.5 mL Eppendorf DNA LoBind tube
Equipment MinION / GridION device、P1000、P200、P20、ice bucket、vortex、microfuge
註釋Note

Wash Mix（WMX）含 DNase I；Wash Diluent（DIL）含 exonuclease buffer，兩者混合後發揮 DNA 消化作用。Storage Buffer（S）用於 wash 完成後的保存，或 reload 前的 flow cell check。

Wash Mix 配製

警告Warning

Wash Mix 建議每次新鮮配製，不要預先配製後存放超過 1 天。WMX 不可 vortex，保持在冰上直到使用。

WMX 解凍方式：從 -20°C 取出後，置冰上解凍；不可室溫解凍。

Reagent 每張 flow cell 用量
Wash Mix（WMX） 2 µL
Wash Diluent（DIL） 398 µL
Total 400 µL

配製步驟：

  1. WMX 置冰上，不 vortex。
  2. DIL 室溫解凍，vortex 混勻後 brief spin-down，再置冰上。
  3. 於 fresh 1.5 mL DNA LoBind tube 中混合 WMX + DIL。
  4. Pipette 混勻（不 vortex），置冰上備用。

Wash / Preserve SOP

1. Stop / pause sequencing run

  1. 在 MinKNOW 中 stop 或 pause sequencing experiment。
  2. 確認 flow cell 不在 sequencing 狀態（manipulating flow cell 時若仍在 sequencing，可能造成 pore loss）。
  3. Flow cell 全程保持插在 sequencing device 上，以維持固定與溫控。

2. 移除 waste buffer

重要Important

移除 waste buffer 前，priming port 與 SpotON sample port 必須為關閉狀態。若兩個 port 在抽 waste 時開著，空氣可能被抽過 sensor array，造成顯著 pore loss。

  1. 確認 priming port cover 與 SpotON sample port cover 皆為關閉狀態。
  2. P1000 設定 1,000 µL。
  3. 將 tip 插入 waste port 1。
  4. 緩慢抽出 waste channel 中的 buffer；若需要可多次重複，直到移除乾淨。

3. 開啟 priming port 並去除氣泡

  1. 順時針滑開 priming port cover。
  2. P1000 先設定為 200 µL，tip 插入 priming port。
  3. 緩慢轉動 wheel，使設定值增加至約 220–230 µL，直到看見少量液體進入 tip（實際吸回量約 20–30 µL）。
  4. 目視確認 priming port 到 sensor array 仍有連續 buffer 覆蓋。
註釋Note

Flow cell 跑過 library 後，priming port 出現 buffer 的速度可能較慢，屬正常現象；耐心等待液體進入 tip，不要過度抽吸。

4. 第一次加入 Wash Mix

  1. P1000 吸取 200 µL prepared Wash Mix，確認 tip 內無氣泡。
  2. 由 priming port 緩慢注入；注入時間至少 5 秒，保持連續速度。
  3. Tip 末端保留少量液體，不要完全打空。
  4. 靜置 5 min。

5. 第二次加入 Wash Mix

  1. 5 min incubation 後，再次吸取 200 µL prepared Wash Mix，由 priming port 緩慢注入。
  2. 關閉 priming port。
  3. 等待 1 hr。
重要Important

1 hr incubation 讓 DNase I 充分消化殘留 DNA library。等待期間不要操作 flow cell。

6. 移除 waste buffer（1 hr 後）

1 hr incubation 完成後：

  1. 開啟 priming port。
  2. P1000 設定 1,000 µL，tip 插入 waste port 1。
  3. 緩慢抽出所有 waste buffer；重複直到移除乾淨。

此時 flow cell 清洗完成，可依需要選擇保存或直接 reload。

7. 加入 Storage Buffer 保存

重要Important

Wash 完成後，不可讓 flow cell 在 Wash Mix 狀態下長時間靜置；這可能造成不可逆 pore loss。若不立即 reload，必須接續加入 Storage Buffer。

  1. 室溫解凍一管 Storage Buffer（S）。
  2. P1000 先設定 200 µL，tip 插入 priming port，緩慢吸回 20–30 µL 去除氣泡。
  3. 取 500 µL Storage Buffer，由 priming port 緩慢注入；注入時間至少 5 秒，保持連續速度。
  4. 關閉 priming port。
  5. P1000 設定 1,000 µL，由 waste port 1 移除所有 waste buffer。
  6. 確認 priming port 與 SpotON sample port cover 皆為關閉狀態。
  7. 將 flow cell 放回原包裝或合適容器，保存於 2–8°C。
註釋Note

保存後取出再使用時，先讓 flow cell 室溫回溫約 5 分鐘，再執行 flow cell check 評估可用 pore 數，確認後再進行 priming 與 loading。

是否適合保存的判斷

條件 建議判斷
剩餘 pores 仍多 保存，後續可用於低風險、探索性或短 run
曾發生明顯 air bubble 保存價值降低；下次使用前特別注意 flow cell check
目標為重要樣本或 publication-grade run 優先使用新 flow cell；reuse flow cell 用於 method development 或補充資料
Flow cell 已多次 wash 預期效能下降；以實際 pore scan / output 決定用途
註釋Note

EXP-WSH004 的一般可重複使用次數為 3–6 次，實際次數取決於樣本種類、前次 run 品質與操作條件。詳細說明請參考 ONT Flow Cell Wash Kit Know-How document。
