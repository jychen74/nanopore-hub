🔬 Kit 選擇決策流程

目標是 16S/ITS 微生物群落鑑定？
├── 是 → 🦠 SQK-MAB114.24（最多 24 樣本，EPI2ME wf-16s）
└── 否（基因組 DNA 定序）
    ├── 需要多樣本 multiplex？
    │   ├── 是 → DNA 量 ≥ 200 ng / 樣本？
    │   │         ├── 是 → HMW + 高純度？
    │   │         │         ├── 是 → 🔬 SQK-NBD114.24（最長讀長）
    │   │         │         └── 否 → 🌱 SQK-RBK114.24（耐受抑制物）
    │   │         └── 否 → DNA ≥ 4 kb 且 1–5 ng 可用？
    │   │                   ├── 是 → 🧪 SQK-RPB114.24（PCR 增幅）
    │   │                   └── 否 → 重新提取
    │   └── 否（單樣本）
    │       ├── 需要最長讀長 / 甲基化 → 🔗 SQK-LSK114（~80 min）
    │       └── 需要最快速 → ⚡ SQK-RAD114（~10 min）
