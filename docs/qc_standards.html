QC 判定標準

Nanopore Sequencing Hub · v1.1 依據 ONT SQK-RBK114.24 / SQK-NBD114.24 官方 protocol 及 ONT Community 建議

Warning

Nanodrop vs Qubit： Nanodrop 同時偵測 RNA、短 fragment、游離核苷酸，濃度值常高估（尤其降解樣本）。本系統以 Qubit dsDNA HS Assay 為準計算投入量。A260/A280 及 A260/A230 比值仍有參考價值但須搭配 Qubit 解讀。

⚗️ Kit 比較：NBD vs RBK

比較項目 SQK-NBD114.24 (Native Barcoding) SQK-RBK114.24 (Rapid Barcoding)
建庫機制 End-prep → Native barcode ligation → Adapter ligation Tagmentation（transposome 切割 + barcode adapter）
建庫時間 約 140 分鐘 約 30–40 分鐘
每樣本 Input 200 fmol（約 130 ng for 1 kb amplicons） 200 ng / 樣本

適用樣本 高品質 gDNA、Amplicon、需超長讀長（SV、phasing） 環境樣本、糞便 metagenomics、哨兵動物樣本、品質次佳樣本

🧮 Kit 推薦評分邏輯

Tip

採評分制而非單一閾值： DNA 品質是多維度的。A260/A230 略低但片段完整且量足夠仍可建庫；A260/A280 完美但嚴重降解 NBD 讀長仍很短。系統綜合評分給出建議，最終決策由操作人員根據實驗目的判斷。

📊 Absorbance Ratio 快速判讀

A260 / A280

  • 🔴 < 1.6 : 嚴重蛋白質或酚類污染
  • 🟠 1.6–1.79 : 輕至中度蛋白污染
  • 🟢 1.8–2.05 : 純 dsDNA 的理想值
  • 🟠 2.05–2.2 : 可能有 RNA 殘留
  • 🔴 > 2.2 : RNA 大量殘留；需 RNase A 處理

A260 / A230

  • 🔴 < 1.2 : 嚴重污染 (腐植酸/GTC/EDTA/多醣)
  • 🔴 1.2–1.49 : 明顯抑制物，建議 AMPure 重純化
  • 🟠 1.5–1.69 : 抑制物殘留，加 BSA 可緩解
  • 🟠 1.7–1.99 : 理想邊緣，NBD ligation 效率略降
  • 🟢 ≥ 2.0 : 理想值

🔬 BSA（牛血清白蛋白）使用指引

  • 作用機制：BSA 吸附腐植酸、單寧酸等酚類化合物及疏水性抑制物，降低其對 transposase / ligase 的抑制效果。對離子性抑制物（鹽類、EDTA）無效。
  • Tagmentation 反應（RBK）：終濃度 0.01 mg/mL
  • Flow cell priming mix：終濃度 0.2 mg/mL (5 µL of 50 mg/mL per 1,170 µL FCF)

(⚠️ 備註：使用 Invitrogen UltraPure BSA (AM2616)，不建議重組人血清白蛋白)
