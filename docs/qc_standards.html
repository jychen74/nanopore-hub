核酸 QC 判定標準

依據 ONT SQK-RBK114.24 / SQK-NBD114.24 / SQK-LSK114 / SQK-RPB114.24 / SQK-RAD114 / SQK-MAB114.24 官方 protocol

警告NanoDrop 與 Qubit 的差異

NanoDrop 同時偵測 RNA、短 fragment、游離核苷酸，濃度值常高估 19–24%（尤其降解樣本）。
本系統以 Qubit dsDNA HS Assay 為準計算投入量。A260/A280 及 A260/A230 仍有參考價值，但不作為投入量計算依據。

三種樣本模式

  • gDNA（高品質基因組）
  • 環境 / 糞便（Metagenomics）
  • 16S/ITS Amplicon（MAB114.24）

適用：組織、細胞株、純培養菌株等高品質 gDNA 樣本

指標 PASS WARNING FAIL
A260/A280 1.80–2.05 1.70–1.79（輕微蛋白污染） < 1.70 或 > 2.10
A260/A230 ≥ 2.00 1.50–1.99（鹽類/有機溶劑殘留） < 1.50
Qubit 濃度 ≥ 20 ng/µL 5–19 ng/µL < 5 ng/µL
Fragment HMW（> 10 kb） Partial（5–10 kb） Degraded（< 5 kb）

最低投入量（依 kit）：

  • SQK-LSK114：1,000 ng（200 fmol，依片段大小換算）
  • SQK-NBD114.24：1,000 ng / 樣本（最多 24 樣本）
  • SQK-RBK114.24：200 ng / 樣本（最多 24 樣本）
  • SQK-RAD114：100–150 ng（單樣本）

建議優先順序： LSK114（單樣本，最大輸出）/ NBD114.24（多樣本，最長讀長）/ RBK114.24（快速備選）

適用：水體過濾膜、糞便、土壤、籠具拭子等環境樣本

指標 PASS WARNING FAIL
A260/A280 1.60–2.20 1.40–1.59 < 1.40
A260/A230 ≥ 1.70 1.20–1.69（腐植酸殘留） < 1.20
Qubit 濃度 ≥ 20 ng/µL 3–19 ng/µL < 3 ng/µL
Fragment HMW 或 Partial Degraded（讀長縮短） 嚴重降解

最低投入量：

  • SQK-RBK114.24：200 ng / 樣本
  • SQK-RPB114.24：1–5 ng / 樣本（需 ≥ 4 kb，PCR 增幅）

建議優先順序： RBK114.24（常規量）/ RPB114.24（超低量 eDNA）

提示

A260/A230 < 1.7 時，建議在 tagmentation reaction 中加入 BSA（終濃度 0.01–0.2 mg/mL） 可部分緩解腐植酸抑制效果。

適用：微生物群落分析，input DNA 僅作為 PCR template，QC 要求較寬鬆。

指標 建議值 注意事項
A260/A280 ≥ 1.8 < 1.8 可能影響 PCR 效率，但耐受度高於 ligation
A260/A230 ≥ 1.5 < 1.5 建議重純化；腐植酸會抑制 LongAmp 聚合酶
Qubit 濃度 可定量即可 只需確認能取到 10 ng
DNA 投入量 10 ng / 樣本 < 12 樣本時 Barcoding 前取 20 ng amplicon
Fragment 長度 無要求 PCR 擴增目標片段，基因組完整性不影響結果

判斷邏輯：

  • A260/A230 ≥ 1.5 → 直接進 PCR
  • A260/A230 < 1.5 → AMPure XP 重純化後再嘗試；若 PCR 產量仍低，可增加 cycles（+3–5）

Fragment Status 判定（凝膠電泳）

等級 判定標準 LSK114 / NBD114.24 fmol 換算
HMW 條帶在 > 10 kb，無明顯瀰散 200 fmol ≈ 2,600 ng（以 20 kb 計）
Partial 主條帶在 5–10 kb 200 fmol ≈ 650–1,300 ng
Degraded 條帶彌散，主要在 < 5 kb 200 fmol ≈ 260 ng（以 2 kb 計）

fmol 換算公式（LSK114 / NBD114.24）：

\[\text{ng} = 200 \times \text{片段大小 (bp)} \times 650 \div 10^6\]

不確定片段大小時，直接用 1,000 ng（保守估計）。
