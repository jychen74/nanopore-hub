# Nanopore Sequencing Hub

This repository serves as a centralized documentation hub for Nanopore sequencing technical resources. It provides standardized protocols and quality control guidelines for laboratory operations.

## Content

* **QC Standards**: Quality control thresholds for DNA samples and environmental/fecal samples, including Nanodrop/Qubit interpretation and BSA usage guidelines.
* **Library Prep SOP**: Detailed standard operating procedures for SQK-RBK114.24 and SQK-NBD114.24 library preparation.
